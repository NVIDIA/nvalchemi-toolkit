  - Navigate: **→ / ↓ / space** next · **← / ↑** back · **Home/End** jump · **`f`** fullscreen ·
    click anywhere = next.
- **`SPEAKER_NOTES.md`** — per-slide talking points + Q&A backstops. `[YOUR PLOT NOTE HERE]`
  marks where your plot annotations go (slides 10 & 12).

## Export to PDF
Open `dd_slides.html`, press `f` (fullscreen), then browser **Print → Save as PDF**
(landscape, background graphics ON). Or ask me to inline the images into a single
self-contained `.html` you can email.
